ui

说明：

依赖模块：'jquery','nojs/nojs.ui'；在扩展ui组件中也需引入该模块

nojs.ui是一个ui工具包，包含最基本的一些核心ui组件和对jQuery的一些扩展工具，基于jQuery1.8

核心ui组件：
ui.align
ui.overlay
ui.layer
ui.popup
ui.msg
ui.select
ui.Switch
ui.slide
ui.menu
ui.ico
jQuery扩展工具：
$.type
$.random
$.stopDefault
$.stopBubble
$.onScroll
$.browser
$.tmpl
$.cookie
$.addCss
